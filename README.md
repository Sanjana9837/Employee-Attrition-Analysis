* NumPy
* Matplotlib

* Scikit-learn

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Encoding
5. Model Training
6. Prediction
7. Model Evaluation

## Result

Achieved high accuracy in predicting employee attrition.
